Welcome! This is the first post on my new blog, built with [Jekyll](https://jekyllrb.com/) and hosted for free on [GitHub Pages](https://pages.github.com/).

## Why Jekyll?

Jekyll is a static site generator — it takes Markdown files and transforms them into a fast, secure website with no database or server-side code required. A few reasons I chose it:

- **Simple**: Write posts in Markdown, commit to git, done.
- **Fast**: Fully static HTML means instant page loads.
- **Free hosting**: GitHub Pages builds and hosts Jekyll sites at no cost.
- **Version controlled**: Every post and change is tracked in git history.
